학생정보관리 프로그램 - 데이터베이스 버전



과제는 Week8-2 영상에 과제 설명이 자세히 되어 있습니다. 





* 제출 파일 : 개발보고서 (포함되어야할 내용: 느낀점, 테이블 구조 설명,  MySQL 쿼리 사용 코드 설명, 추가점수 구현 시 해당 내용 설명 포함)

 - 제출 형식 - DBP-과제2-개발보고서-이름-학번.hwp (doc, pdf..)

 - 추가점수 구현시 제출 형식 - DBP-과제5-개발보고서-이름-학번-추가점수.hwp (doc, pdf..)



지난번 과제와 마찬가지로 추가 점수가 부여됩니다. 
기능은 지난 과제와 전부 동일하며, 데이터 베이스를 이용해서 구현하면 됩니다. 
추가 점수 내용을 구현한 경우, 해당 내용애 대한 자세한 설명도 개발보고서에 포함되어야 합니다.

* 추가 점수 기능에 대한 자세한 설명입니다. 아래 4가지 모두 구현한 경우에 한해 추가 5점이 부여됩니다.

  (연장 제출경우라도 추가 점수 구현시 5점 부여는 동일합니다)

1. 프로그램 시작시 자동 학생 정보 로드 기능 (CheckBox 컨트롤 이용)

2. 현재 저장된 학생 수 표시 (학생은 최대 3명까지만 기록)

3. 학생별 메모 기능 추가 ( 텍스트 여러줄 입력 가능 (Multiline))

4. 학생별 삭제 기능 추가
